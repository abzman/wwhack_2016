# Pinouts #

## Motors ##

### Main Drive ##
| Wire  | Function          |
|:------|:------------------|
| Blue  | (-) for forward   |
| Brown | (+) for forward   |

### Steering ##
| Wire  | Function          |
|:------|:------------------|
| Blue  | (-) for right     |
| Brown | (+) for right     |

## Sensors ##

### LIDAR ###
| Pin   | Function |
|:------|:---------|
| 1     | VCC      |
| 2     | NC       |
| 3     | NC       |
| 4     | SCL      |
| 5     | SDA      |
| 6     | Ground   |
